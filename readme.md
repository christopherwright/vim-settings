# Vim Settings

## Usage

To install:

    $ rake install

**Warning:** This will blast away your existing vim settings.

## Key bindings

**,s**&nbsp;&nbsp;&nbsp;Save  
**,d**&nbsp;&nbsp;&nbsp;Save and quit  
**,x**&nbsp;&nbsp;&nbsp;Quit without saving
